# 國中單字王

這是一個純靜態 PWA 單字學習 App，可部署到 GitHub Pages。使用者可以用 iPhone Safari 開啟後「加入主畫面」，作為類 App 的學習工具使用。

## 網址

- 99 單字版：`https://xier9.github.io/Recite2000SingleWords/`
- 2000 單字版：`https://xier9.github.io/Recite2000SingleWords/2000/`

## 本機預覽

```bash
python3 -m http.server 4173
```

- 99 單字版：`http://localhost:4173/`
- 2000 單字版：`http://localhost:4173/2000/`

## iPhone 使用

1. 用 iPhone Safari 開啟上方網址。
2. 點分享按鈕，選「加入主畫面」。
3. 從主畫面開啟後，可使用已快取的頁面與資料。

## 99 單字版功能

- 學習：一次顯示 5 個單字、中文、詞性、音標與例句。
- 朗讀：點單字、播放按鈕或例句可使用英文語音朗讀。
- 複習：翻卡模式，可標記「已會」或「加強」。
- 測驗：選擇題練習，會優先抽加強與尚未熟悉的字。
- 進度：使用瀏覽器 `localStorage` 儲存在同一台裝置上。

## 2000 單字版功能

- 學習：依 30 個主題分類，不限制單次顯示字數。
- 複習：以標注「加強」的單字為主，可依主題篩選；字卡每輪亂數出現，整輪完成後重新洗牌。
- 測驗：從已會與加強單字出題，提供 20 題測驗與連續練習。
- 成績：20 題測驗每題 5 分，記錄分數、使用時間與答錯單字。
- 分享：可將當次測驗結果文字分享到 LINE。
- 備份：可匯出與匯入已會、加強與 20 題測驗紀錄。

## 安全與隱私

- App 沒有後端伺服器、登入系統或資料庫。
- 學習紀錄只存在使用者裝置的 `localStorage`，除非使用者主動分享或匯出。
- 匯入紀錄只接受 JSON 檔，並限制檔案大小與資料格式。
- 頁面加入基本 Content Security Policy，降低外部腳本注入風險。
- Service worker 只快取同網域且成功載入的靜態資源。
